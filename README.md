## Как добавить новый пост
1. Заходим в папку `_posts/`
2. Создаём файл с именем `ГГГГ-ММ-ДД-korotkiy-nazvanie.md`
   Пример: `2025-05-01-subbotnik.md`
3. В начале файла пишем:
```

---

title: "Название поста"
date: 2025-05-01

---
   А дальше — сам текст в простом формате.
4. Сохраняем, заливаем на GitHub
